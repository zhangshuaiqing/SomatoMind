## 快速开始

```bash
# 克隆
git clone https://github.com/zhangshuaiqing/SomatoMind.git
cd SomatoMind

# 安装依赖
pip install -r requirements.txt  # 或 uv sync

# 运行实验
python experiments/e1_reflection/run.py
```
