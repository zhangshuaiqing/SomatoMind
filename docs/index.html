📋 认知架构详述 / Architecture Breakdown

层级模块描述数据流方向
⚡ Fast Loop
ms — s		 Perception将环境观测转换为内部表征。包含感知适配器（归一化各类输入）、注意力机制（筛选关键特征）、世界模型（前向预测）→ Memory + Cognition
Memory System四层级记忆：Sensory Buffer（毫秒原始数据）、Working Memory（秒级任务上下文）、Episodic Memory（分钟级轨迹）、Procedural Memory（永久技能）↔ Cognition
Cognition六种高级认知模块：Reasoning（符号+概率推理）、Planning（多步搜索+目标分解）、Reflection（元认知反思）、Spatial（空间推理+心理模拟）、Creativity（探索+反事实）、Policy（动作选择）→ Action
Action将决策映射为环境可执行的动作，包含策略桥接、运动控制和反馈闭环→ Environment
🧠 Slow Loop
s — min		 Environment环境的反馈回路。支持 GridWorld（full/local/fog 观测）、动态障碍物、多 Agent Stigmergy、ROS2+Gazebo 仿真⇢ Perception (feedback)
Reflection慢回路核心：从 Episodic Memory 调取轨迹 → 推理评估 → 规划调整 → 修正快回路行为⇢ Cognition (modify)
🔍 Meta Layer
min — hour		 Meta Monitor监控整体表现指标，当成功率低于阈值时触发自我修改。同时评估反思质量——"反思是否真正改变了行为？"→ Plasticity
Plasticity部署后自我进化的三种机制：EWC（保护重要权重防遗忘）、Hebbian（局部学习规则）、Structural（动态增删神经元或连接）→ Brain (structural)
